# TarbiyahGuide - Islamic Parenting Resource

A comprehensive website dedicated to raising Muslim children according to the Quran and Sunnah, with a special focus on raising daughters and marriage preparation.

## Features

- **General Tarbiyah**: Core principles of Islamic parenting, practical tips, and common pitfalls.
- **Raising Daughters**: A dedicated section on the immense rewards and holistic upbringing of daughters.
- **Marriage Preparation**: Guidance on choosing a spouse, the importance of consent, and parental duties.
- **Resources**: Curated list of books and lectures by scholars like Dr. Yasir Qadhi.

## Tech Stack

- React (Vite)
- Tailwind CSS (v4)
- React Router DOM
- Lucide React (Icons)

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```

## Design

The site features a premium "Emerald & Gold" aesthetic, reflecting Islamic artistic traditions with a modern, clean interface.
